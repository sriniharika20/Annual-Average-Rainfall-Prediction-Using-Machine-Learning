Annual Average Rainfall Prediction Using Machine Learning

This project predicts the annual average rainfall based on meteorological factors such as temperature, humidity, wind speed, and visibility using a Random Forest Regressor model.
It also provides a simple and interactive Gradio web interface for making predictions directly from user input.
